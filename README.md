# Personal-Site

Achievement-focused personal website with a premium, recruiter-friendly UI.

## Run locally

Open `index.html` directly in a browser, or serve with any static server.

## Customize

- Update contact links in `index.html` (`#contact` section).
- Edit starter achievement data in `script.js` (`seedAchievements`).
- Add new achievements from the live UI using the `Add Achievement` button.
